Linux basic commands

1. whoami to show which user we are
2. pwd present working directory where i am working right now.
3. ls is to list the content present
4. cat command is used to read the file content
5. sudo - i to switch to root user home directory
6. cd / change to root directory
7. mkdir to make directory
8. touch to create file
9. cp to copy file from one location to another.
10. cp -r to copy one directory to another directory.
11. mv is used to move file from one directory to another it is also used to rename file.
12. rm is used to remove file  rm -r to remove directory
13. * means everything
